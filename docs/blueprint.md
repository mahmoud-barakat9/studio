# **App Name**: Abjour Track

## Core Features:

- User Order Creation: Allow users to create new abjour orders by providing their name, contact number, and other necessary information.
- Order Specification: Enable users to specify details for each abjour opening, including serial number, abjour type, color, code length, and number of codes, and calculate length/number of codes based on width/type using AI.
- Order Tracking: Provide a visual order tracking system for users to monitor the status of their orders, from creation to completion.
- Order Naming: Allow customers to give individual names to each order, for easier accounting.
- Automated Cost and Space Calculation: Calculate the total cost and space required for each order automatically, visible to both the customer and admin. Only the admin can use a tool to edit pricing and calculations.
- Admin Order Management: Enable administrators to create and manage orders, adjust prices, and oversee the entire order process.
- User Management: Allow administrators to manage user accounts, roles, and permissions.
- Reporting: Generate reports on order volume, revenue, and other key metrics.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5), evoking reliability and professionalism, key in managing important orders.
- Background color: Light blue-gray (#E8EAF6), desaturated and light for a clean, calming backdrop.
- Accent color: Indigo (#5C6BC0), used for interactive elements, provides a visually distinct contrast without disrupting the calm aesthetic.
- Body and headline font: 'Inter', a sans-serif typeface that balances approachability with precision; suited to displaying large quantities of data in a readable fashion.
- Use clean, modern icons to represent different order statuses and actions.
- Design a clean and intuitive layout for both desktop and mobile platforms.
- Incorporate subtle animations to enhance user experience and provide feedback on interactions.
